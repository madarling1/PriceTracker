#### 마비노기 모바일의 영혼석, 마력석, 세공기의 거래소 시세를 모니터링합니다.<br>

# 다운로드
1. [Releases](../../releases)를 엽니다.
2. PriceTracker.exe를 다운로드 합니다.
3. 다운로드한 'PriceTracker.exe'를 실행합니다.
> 💡 보안 경고가 뜨면 **"추가 정보" → "실행"** 을 눌러주세요. (바이러스 아니에요!)<br>
> 💡 또는 **"점 세개" → "유지" → "삭제버튼 우측 드롭다운 클릭" → "그래도 계속"** 을 눌러주세요.
